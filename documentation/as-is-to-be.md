Выявленные проблемы

Проблемы и последствия:

Потеря задач в чатах - Пропущенные дедлайны, конфликты
Ручной ввод реквизитов - Ошибки переводов, фрод
Отсутствие единого трекера - Невозможно отследить прогресс
Ручная сверка платежей - Затраты времени бухгалтера 4-6 часов/неделю, возможность ошибок
Финансовая отчетность - Нет точной информации по выручке
